# 🏋️ Gym Customer Churn Analysis Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze customer churn in a gym business. The dashboard provides insights into customer behavior, membership trends, revenue, attendance, and factors influencing customer retention.

The objective is to help gym owners make data-driven decisions to improve customer satisfaction and reduce churn.

---

## 🎯 Business Problem

Customer churn directly affects the profitability of fitness centers. Understanding why members leave and identifying at-risk customers can help improve retention strategies.

This dashboard answers questions like:

- What is the overall churn rate?
- Which membership plans have the highest churn?
- Which age groups are more likely to leave?
- How does attendance affect churn?
- What factors contribute most to customer retention?
